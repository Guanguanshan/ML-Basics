# 第 3 章 · 线性模型（周志华《机器学习》）

本目录对应教材 **第 3 章**，用 **NumPy + Matplotlib** 手写或拼装核心流程，侧重公式与梯度、闭式解的对照，以及可视化与「理想 / 非理想」数据上的行为差异。

---

## 文件一览

| 文件 | 复现的模型 / 内容 | 做了什么实验 | 特点 | 产出图片 | 图片含义 |
|------|-------------------|--------------|------|----------|----------|
| `ch3_2.py` | **一元线性回归**（MSE + 梯度下降） | 在 5 个 toy 点上迭代更新 \(w,b\)，每隔若干 epoch 叠加拟合直线 | 仅依赖 `numpy`、`matplotlib`；显式写出 MSE 对 \(w,b\) 的梯度 | `ch3_2_fit.png` | 同一散点图上多条直线，展示**训练过程中拟合线如何逐渐贴近数据** |
| `ch3_3.py` | **对数几率回归 / Logistic 回归**（二分类） | 一维特征、0/1 标签；交叉熵 + sigmoid + 梯度下降；打印 loss 曲线数值 | 手写 sigmoid 与交叉熵梯度；`log` 加小常数防数值问题 | `ch3_3_logistic.png` | 散点 + **sigmoid 概率曲线**，表示决策边界在特征轴上的「软」过渡 |
| `ch3_4.py` | **线性判别分析 LDA**（二分类） | 两组二维高斯样本：**理想**（可分性好）vs **非理想**（双峰、协方差差异等） | 手写 \(S_w\)、投影方向 \(w\)、阈值；`train_lda_by_lagrange` 供 `ch3_5` 复用；含投影与决策边界可视化 | `ch3_4_lda.png` | 左右两子图对比 **LDA 在「类条件近似高斯」与复杂分布下的分界与准确率** |
| `ch3_5.py` | **多分类**：在 LDA 二分类器上用 **OvR / OvO** | 两套 3 类二维数据：① 加噪 + 离群；② 非高斯、弯月等**非理想**结构；对比两种分解策略的训练准确率与散点着色预测 | 依赖 `from ch3_4 import train_lda_by_lagrange`；强调 **线性假设被破坏时** OvR/OvO 仍只是二分类器组合，不能变出非线性边界 | `ch3_5_multiclass.png` | 2×3 子图：**真实标签 / OvR 预测 / OvO 预测** ×（原有数据 \| 非理想数据），直观看出 **哪类数据上多分类 LDA 退化** |
