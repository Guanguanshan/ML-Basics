# 第 4 章 · 决策树（周志华《机器学习》）

本目录对应教材 **第 4 章**：**离散属性 ID3 风格决策树**（划分准则可插拔）、**剪枝**、以及教材 **表 4.3 / 表 4.4** 上的**连续属性二分**与**缺失值加权划分**。数据以西瓜书表格为主，图用于对照「不同准则 / 是否剪枝 / 混合特征」下的树形与准确率。

---

## 文件一览

| 文件 | 复现的模型 / 内容 | 做了什么实验 | 特点 | 产出图片 | 图片含义 |
|------|-------------------|--------------|------|----------|----------|
| `ch4_1.py` | **决策树分类器骨架**（递归建树 + 预测） | 无单独数据集实验；为 `ch4_2`–`ch4_4` 提供 `DecisionTreeClassifier`、`predict_from_node` 等 | **不内置**划分准则：通过 `choose_best_attr` 注入信息增益 / 基尼等；支持 `max_depth`、`min_samples_split`、训练样本 id 记录（供后剪枝） | （无） | 库模块；树结构由调用方传入的准则函数决定 |
| `ch4_2.py` | **离散决策树 + 三种划分准则** | **西瓜数据集 2.0**（表 4.1）：分别用 **信息增益、增益率、基尼指数** 建树，并在训练集上算准确率 | 三种 `choose_best_*` 与同一套 `ch4_1` 树算法组合；自绘树布局保存 PNG | `ch4_2_trees.png` | **并排三棵树**：对比同一数据下 **根结点与分支结构** 如何随准则变化（与教材讨论「信息增益偏好多取值」等呼应） |
| `ch4_3.py` | **预剪枝 + 后剪枝** | 仍用西瓜 2.0：**无剪枝全树** vs **`max_depth=2` 预剪枝** vs **在 12 训 / 5 验分层划分上的后剪枝**；子图标题中带训练 acc 与**同一验证集** acc | 后剪枝用验证集到达结点统计误差，**仅当叶结点在验证集上严格优于子树**才剪，避免平局乱剪 | `ch4_3_prune.png` | **三棵树并列**：展示 **过拟合全树** 与 **两种剪枝** 在复杂度与验证表现上的折中（小样本下验证划分方式见代码注释） |
| `ch4_4.py` | **混合属性树**：离散 + **连续二分**；离散上的 **缺失值（加权样本）** | **表 4.4** 西瓜 2.0α（`None` 表示缺失）按 §4.4.2 思路建树并预测；**表 4.3** 西瓜 3.0 含密度、含糖率，对连续特征做候选阈值的信息增益二分 | 与 `ch4_1` 平行的一套 `_build_hybrid` 逻辑（非同一类名），支持连续结点 `≤t` / `>t` 与缺失分支权重 | `ch4_4_hybrid.png` | **左右两子图**：左为 **仅离散 + 缺失** 的树与训练 acc，右为 **离散 + 连续** 的树与 acc，对应教材 **4.4 连续与缺失值** 两节的可视化复盘 |
